# Ghost brand family

One ghost, many moods — the shared mascot for [interesting-vibe-coding](https://github.com/interesting-vibe-coding).

| Variant | File | Use |
|---------|------|-----|
| Terminal | `ghost-terminal.svg` | Helm app icon — round eyes + `>_` prompt mouth |
| Waving | `ghost-waving.svg` | Org avatar — friendly, no terminal element |
| Wink | `ghost-wink.svg` | Screenshots / social — playful, keeps the `>_` mouth |

Palette: dark rounded base `#16131f → #0c0a12`, purple ghost `#c4a3f5 → #9b6dff`.
